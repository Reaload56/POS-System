# Retail POS & Inventory Management System

A desktop **Point of Sale (POS) and Retail Management System** built using **C#, WinForms, and .NET**.  
The system is designed to help small retail businesses manage products, inventory, and sales through a simple desktop interface.

This project demonstrates practical implementation of **CRUD operations, authentication systems, and retail transaction workflows** in a Windows desktop environment.

---

## Overview

The POS System provides an integrated solution for managing retail operations including:

- Secure login access
- Product management
- Inventory tracking
- Sales transactions

The application was built using **Microsoft Visual Studio** and is intended as a learning project demonstrating desktop application development with **C# and WinForms**.

---

## Features

- 🔐 **User Authentication**
  - Login system to restrict system access

- 📦 **Product Management**
  - Add new products
  - Update product information
  - Delete products
  - View product list

- 📊 **Inventory Management**
  - Track product stock levels
  - Update inventory records

- 🧾 **Sales / POS Interface**
  - Process customer purchases
  - Display available products dynamically

- 🖥 **Desktop Interface**
  - Built with Windows Forms for a responsive UI
